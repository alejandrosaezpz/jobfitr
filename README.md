# JobFitr

JobFitr es un repositorio base para una futura plataforma de apoyo a procesos de empleo y matching de perfiles.

## Estado actual

Este repositorio contiene únicamente la estructura documental y organizativa aprobada.
No hay backend, frontend ni lógica de negocio funcional todavía.

## Qué incluye

- Documentación base en `docs/`
- Estructura inicial para backend y frontend
- Archivos de soporte: `.gitignore`, `.env.example`, `requirements.txt`, `LICENSE`

## Estructura

- `backend/app/` — base del backend
- `frontend/src/` — base del frontend
- `tests/` — espacio para pruebas futuras
- `.github/workflows/` — automatización futura

## Nota honesta

Los nombres y ejemplos incluidos en esta base son orientativos y ficticios cuando corresponde.
No deben interpretarse como funcionalidad ya implementada.
